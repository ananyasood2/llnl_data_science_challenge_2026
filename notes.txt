The important concept:
MCP is not the AI.
MCP is the standardized bridge that lets an AI agent interact with external tools.
Task 2 preview
Task 2 is where the challenge starts becoming more "agentic".
Task 1:
"Give the agent one tool."

Task 2:
"Give the agent the ability to analyze results."

You will probably add a tool that extracts statistics from the segmented CT data, such as:
volume fraction
number of connected components
voxel statistics
geometric information
The workflow becomes:
Codex
 |
 +---- segment_ct_dataset()
 |
 +---- analyze_segmentation()
 |
 +---- generate_report()
The agent starts chaining tools together.


Questions scientists want answers too ->

They want answers like:
How much of the material exists?
What percentage of the volume is occupied?
How many disconnected structures are there?
What is the geometry of the material?
